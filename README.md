# Smart Gate Pass System

A Smart Gate Pass System designed to automate campus entry and exit using **Facial Recognition** and **Encrypted QR Verification**. The platform provides **role-based dashboards** for Students, Faculty, Admin, and Security Officials, ensuring secure, transparent, and streamlined movement tracking inside the campus.

##  Overview
This AI-powered system integrates **Deep Learning-based Facial Recognition** and **role-based workflow automation** to securely verify identities and manage gate-pass approvals. It reduces manual workload, prevents impersonation, and enables real-time monitoring.
